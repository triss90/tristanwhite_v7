What is it?

Colodrop is yet another color combination webapp. It lets you explore hundreds of different color combinations and even allows you to save your favourite colors.

The Frontend

The frontend is generated from a custom built PHP micro-framework, that easily intergrates with the backend to allow for highly costumized data layouts.

The Backend

Colodrops backend is simple, yet allows for unlimited future expansion and cosutmization. Currently it only offers the ability to add new colors. It's built on PHP and utilizes JS for seamless database queries.
